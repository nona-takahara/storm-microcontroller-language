## BOOL_FUNC_4（Boolean Function (4 inputs)）

> ヒント（実機で驚きうる点。答えられるものだけで可。分からなければ本文は空欄のままでOK）:
> - プロパティの値と入出力の対応関係が、DSLに書いた設定通りに実機へ反映されるか

Stormworks JP Wikiより転載。※Operatorはゲーム内では半角の記号を使用してください。ここではWikiの仕様で全角で表記しています。

Variables(変数)	補足
x	Input	入力値
xの他、それぞれのパーツごとに定義された変数名(y,z,wなど)が式の中で使用できる。
Constants(定数)	補足
true	True(真)	真
false	False(偽)	偽
Operators(演算子)	補足
＆	And	論理積
｜	Or	論理和
＾	Xor	排他的論理和
x^yは、(x＆！y)｜(！x＆y)と等しい。
同じ記号が算術式で累乗の演算子に使用される。
！	Not	否定論理
変数の前（！x）または後ろ（ｘ！）に付けられる。たとえば「『xが真』かつ『yが偽』かつ『zが真』」ならば「x＆！y＆z」「x＆y！＆z」のどちらでも使える。
